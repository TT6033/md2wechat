# 更新日志

## [1.1.0] - 2026-06-19

### ✨ 新增

- **可视化样式面板** — 颜色选择器、字号/行高/字间距滑块、段落对齐、字体族、H2 风格下拉，选择即生效
- **10 套内置模板** — 新增科技暗黑、文艺范、粉色少女、中国红、学术严谨 5 套模板
- **随机配色** — 一键随机生成配色方案
- **方案保存/加载** — 将样式配置保存为 JSON，支持导入导出和本地存储
- **颜色选择器** — 主色和辅色支持颜色选择器 + 十六进制输入双模式
- **H2 多种样式** — 左边框竖线、居中下划线、渐变背景、底部粗线、纯文字
- **多字体族** — 系统默认、衬线、无衬线、等宽

### 🔄 改进

- 移除 CSS 代码编辑器，改为全可视化控件操作
- 模板选择条改为 Chip 样式，更直观
- 预览信息显示当前模板名称

## [1.0.0] - 2026-06-19

### ✨ 初始版本

- Markdown 实时预览，左侧编辑右侧渲染
- 5 套基础配色主题（微信绿、商务蓝、优雅紫、活力橙、极简灰）
- 字号调节（12px ~ 22px）
- 所有样式自动转为 inline，兼容微信公众号编辑器
- 导入 .md 文件
- 导出 HTML 文件（含代码高亮）
- 一键复制富文本到剪贴板
- 快捷键 `Cmd/Ctrl+S` 复制
- marked.js 解析 Markdown，highlight.js 代码高亮
- 纯前端单文件，无后端依赖
